DensePose

Human pose estimation has been a popular research topic in the past few years. Machines are able to recover human skeleton structure by estimating 3d human key points. However, key points are one kind of sparse representations. Sometimes we want a more dense representation. This paper gives a possible solution towards predicting dense correspondences from an image to a surface-based model.

Overview

  • Gather dense correspondences for 50K persons appearing in the COCO dataset by introducing an effieient annotation pipeline.
  • Use this dataset to train CNN-based systems that deliver dense correspondences ‘in the wild’, by regressing body surface coordinates at any image pixel.
  • Use sparse correspondences to train a ‘teacher’ network that can ‘inpaint’ the supervision signal in the rest of the image domain. This gain better performance when compared to either sparse points or any other existing dataset.

DATASET

  • Called COCO-Dense Pose.

  • UV map defination:

    • Head, hands, feet: SMPL defined UV map
    • Other parts: obtain the unwrapping via multidimensional scaling applied to pairwise geodesic distances.

  • Annotation: 1. Region; 2. Sparse correspondences

Network

Region-based dense pose regression

Multitask cascaded architectures

Learning

  • Even though the target is dense pose estimation at test time, in every training sample, the dataset only annotate a sparse subset of the pixels, approximately 100-150 per human.
  • This is not a problem during training, the solution is to making classification/regression losses oblivious to points where the group-truth correspondence was not collected, simply but not including them in the summation over the per-pixel losses.
  • By ‘inpainting’ the values of the supervision signal on positions that were not originally annotated, the network botain better results. For this, the author adopts a learning-based approach where they firstly train a ‘teacher’ network to reconstruct the ground-truth values wherever these are observed, and then deploy it on the full image domain, yielding a dense supervision signal.
